Two Column Layout Examples

Layout Variations

This page demonstrates two common two-column layout patterns using CSS Grid.

Try it: Resize your browser window to see how the layouts adapt to different screen sizes!

Equal Columns Layout (1fr 1fr)

Left Column

This is the left column content. It takes up exactly half of the available space on desktop screens.

Equal column layouts are perfect for:

  • Comparing two items side by side
  • Before/after content
  • Feature comparisons
  • Team member profiles

On mobile devices, this column will stack above the right column for better readability.

Right Column

This is the right column content. It also takes up exactly half of the available space.

The CSS Grid property grid-template-columns: 1fr 1fr creates two equal columns that automatically adjust to the container width.

Key benefits:

  • ✅ Automatic equal sizing
  • ✅ Responsive behavior
  • ✅ Clean visual balance
  • ✅ Easy to implement

Content + Sidebar Layout (2fr 1fr)

Main Article Content

This is the main content area that takes up about 67% (2/3) of the available space. It's perfect for blog posts, articles, or any primary content.

This layout uses grid-template-columns: 2fr 1fr which creates a 2:1 ratio between the main content and sidebar.

When to Use This Layout:

  • Blog posts with sidebar widgets
  • Product pages with related items
  • Articles with table of contents
  • News sites with advertisement space

The layout automatically adapts to different screen sizes, stacking the sidebar below the content on mobile devices for optimal readability.

CSS Grid Benefits:

Using CSS Grid for this layout provides several advantages over traditional float or flexbox approaches:

  • Precise control over column ratios
  • Built-in gap control
  • Easy responsive behavior
  • No clearfix hacks needed
